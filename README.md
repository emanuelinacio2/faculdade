# Projeto de Software

Projeto em Next.js para guardar conteudos da materia de Projeto de Software. Os
dados ficam no `localStorage` do navegador.

## Paginas

- Pagina inicial com resumo e atalhos.
- Pagina para adicionar conteudos.
- Pagina para listar conteudos salvos.

## Rodar localmente

```bash
npm run dev
```

## Build

```bash
npm run build
```
