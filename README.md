# ADS Conteudos

Projeto em Next.js para guardar conteudos da faculdade de ADS. Os dados ficam no
`localStorage` do navegador.

## Paginas

- Pagina inicial com resumo e atalhos.
- Pagina para adicionar conteudos.
- Pagina para listar conteudos salvos.

## Rodar localmente

```bash
npm run dev
```

## Build

```bash
npm run build
```
